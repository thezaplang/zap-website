## Installation

### Prerequisites

- Git
- CMake >= 3.31.6
- Make/Ninja
- Clang >= 19
- LLVM >= 19

## clone repository

```bash
git clone https://github.com/thezaplang/zap.git
cd zap
chmod +x build.sh
./build.sh
```

### Verify Installation

```bash
zapc --version
```
